# Goal Setting Backend

The goal setting app is powered by a Flask backend. 
SQLAlchemy database is used to store user's goals. The backend uses sentiment analysis to get a numerical rating on user's journal entries. User's may also evaluate their progress on their goals
which is ran by OpenAI API supported agent. 

User's can also chat with a virtual therapist bot which uses whispr and other text-to-speech libraries in python.
